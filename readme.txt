usage:

if you have access to sites.json, run sh sitesjson2zitedownload.sh or sh sitesjson2zitelist.txt in /path/to/zeronet/data/ and you will get a list

if you don't have access to sites.json, open zerohello in firefox browser and press F12, then go to the console and type zeroframe.cmd("siteList") then, copy the object and save it as return.txt and run sh sitelist2zitedownload.sh or sh sitelist2zitelist.sh

so, what is a zitelist used for?

you can use it to download zites to your zeronet client. zitelists are usually getten from public proxies which usually have a lot of zites on them. and it's like copying a public proxy to your own hard disk.
